## 🚀 빠른 시작 가이드 (10분 완성)

1.  **시스템 접속**: 구글 계정으로 로그인하여 나만의 대시보드를 생성합니다.
2.  **AI 두뇌 연결**: `설정` 메뉴에서 API 키를 입력합니다.
3.  **황금 키워드 발굴**: `골든시스템`에서 노출 확률이 높은 주제를 선택합니다.
4.  **콘텐츠 발행**: `수익형 콘텐츠 생성`에서 원고를 만든 뒤 블로그에 포스팅합니다.

---

## 💰 수익화 흐름 설명

1.  **STEP 1. 콘텐츠 생성**: AI가 SEO 최적화된 전문 원고를 작성합니다.
2.  **STEP 2. 노출 및 유입**: 검색 엔진 상단에 노출되어 고객이 내 글을 읽습니다.
3.  **STEP 3. 가망고객 수집**: 글 하단의 랜딩페이지 링크를 통해 고객이 상담 신청을 남깁니다.
4.  **STEP 4. 실시간 관리**: 유입된 고객 DB를 대시보드에서 즉시 확인합니다.
5.  **STEP 5. AI 솔루션 상담**: AI 분석 제안서를 활용해 계약을 체결합니다.

## 📊 메뉴별 상세 매뉴얼

### 1. 골든시스템 (Golden System)
*   **✔ 목적**: 상위 노출을 위한 '황금 키워드'와 '최적화 주제' 발굴
*   **✔ 사용 방법**: 키워드 입력 -> 분석하기 -> 주제 선택
*   **✔ 실제 예시**: '치아보험' 입력 시 -> '임플란트 보장 많이 받는 법' 제안

### 2. 수익형 콘텐츠 생성 (AI Marketing)
*   **✔ 목적**: 가독성 디자인이 적용된 블로그 원고 및 이미지 자동 생성
*   **✔ 시스템 특징**: 5단 구조, 문장 줄바꿈, 핵심 단어 강조, AI 이미지 자동 삽입
*   **✔ 사용 방법**: 주제 선택 -> 스타일 설정 -> 자동생성 -> 서식포함 복사

### 3. 가망고객 수집 (Landing Page)
*   **✔ 목적**: 상담 신청률을 극대화하는 랜딩페이지 카피 생성
*   **✔ 사용 방법**: 업종 및 혜택 입력 -> 랜딩페이지 생성

### 4. 보험설계 솔루션 (AI Solution)
*   **✔ 목적**: 고객 맞춤형 3가지 설계안(A/B/C) 시각화 리포트 제공
*   **✔ 사용 방법**: 고객 요구사항 입력 -> AI 분석 시작 -> 리포트 공유

### 5. 유용한 기능 (Useful Tools)
*   **명함제작**: 5종의 세련된 템플릿 제공
*   **질병코드분석**: KCD 코드 기반 보장 분석
*   **금소법 가이드**: AI 법률 문구 검토
*   **보험나이 계산기**: 상령일 자동 계산
